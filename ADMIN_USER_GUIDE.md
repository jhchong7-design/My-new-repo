## ✏️ Using the Inline Editor

### Getting Started
1. Navigate to any page on the website
2. Locate the pencil/edit icon (bottom-right, 120px from bottom)
3. Click the icon to enter edit mode
4. If not logged in, you'll be prompted to login

### Editing Content

#### Text Editing
1. **Click on any text element** to make it editable
2. **Type your changes** directly on the page
3. **Auto-save** activates automatically after 2 seconds
4. **Watch for the green checkmark** indicating save completion

#### Rich Text Editing
For more advanced formatting:
1. Double-click on a block to open the rich text editor
2. Use the formatting toolbar to:
   - Bold, italic, underline
   - Create lists (bulleted/numbered)
   - Add links
   - Insert images
   - Format headings
3. Click "Save" or press Ctrl+S to save changes

### Block Actions
Each editable block has action buttons:

**📌 Move Up/Down**
- Click the up arrow to move the block above
- Click the down arrow to move the block below
- Useful for reordering content sections

**🗑️ Delete**
- Click the trash icon to remove the block
- You'll be asked to confirm deletion
- Deleted blocks cannot be recovered

### Drag-and-Drop Reordering
1. locate the drag handle (grip icon) on the left of each block
2. Click and hold the handle
3. Drag the block to your desired position
4. Release to drop it in the new location
5. Changes are auto-saved

### Preview Mode
To preview your changes as visitors would see them:
1. Look for the "Preview" toggle button (top of page)
2. Click to switch to preview mode
3. Content displays without edit controls
4. Click again to return to edit mode

### Keyboard Shortcuts
Speed up your editing with these shortcuts:

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` (or `Cmd+S`) | Save changes immediately |
| `Escape` | Exit edit mode |
| `Ctrl+Z` (or `Cmd+Z`) | Undo last change |
| `Ctrl+Y` (or `Cmd+Shift+Z`) | Redo undone change |

## 🎯 Best Practices

### ContentEditing
1. **Preview before publishing** - Always check how your changes look
2. **Save frequently** - Let auto-save work, but manually save important changes
3. **Use meaningful headings** - Organize content with clear structure
4. **Add alt text to images** - Improves accessibility and SEO
5. **Check mobile view** - Ensure content looks good on all devices

### Security
1. **Logout after editing** - Especially on shared computers
2. **Keep credentials secure** - Don't share your login details
3. **Use strong passwords** - Update passwords regularly
4. **Monitor activity** - Check the activity log regularly

### Performance
1. **Optimize images** - Compress images before uploading
2. **Use appropriate formats** - JPEG for photos, PNG for graphics
3. **Keep pages lightweight** - Avoid excessive content

## 🔧 Troubleshooting

### Editor Doesn't Appear
**Problem:** Pencil button not visible

**Solutions:**
1. Refresh the page
2. Clear browser cache
3. Check that JavaScript is enabled
4. Try a different browser
5. Check browser console for errors (F12)

### Can't Edit Content
**Problem:** Clicking on content doesn't make it editable

**Solutions:**
1. Verify you're logged in
2. Check that you're in edit mode
3. Try refreshing the page
4. Check your user permissions

### Auto-Save Not Working
**Problem:** Changes aren't saving automatically

**Solutions:**
1. Wait 2 seconds for auto-save to trigger
2. Look for the green checkmark indicator
3. Check your internet connection
4. Manually save with Ctrl+S
5. Check server status

### Changes Not Persisting
**Problem:** Changes disappear after refresh

**Solutions:**
1. Verify the save completed (green checkmark)
2. Check your internet connection
3. Verify you have write permissions
4. Check browser console for errors
5. Contact technical support

### Dashboard Not Loading
**Problem:** Admin dashboard won't open

**Solutions:**
1. Clear browser cache and cookies
2. Try a different browser
3. Check if server is running
4. Verify your login credentials
5. Check network connectivity
