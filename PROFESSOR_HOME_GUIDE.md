# 강성열 교수 홈페이지 - 사용자 가이드

## 📋 개요

강성열 교수 홈페이지는 관리자가 정보를 추가, 변경, 삭제할 수 있는 종합적인 관리 시스템을 갖춘 홈페이지입니다.

- **메인 페이지**: https://00u3b.app.super.myninja.ai/professor-main.html
- **로그인**: https://00u3b.app.super.myninja.ai/professor-login.html
- **회원가입**: https://00u3b.app.super.myninja.ai/professor-register.html
- **관리자 대시보드**: https://00u3b.app.super.myninja.ai/professor-admin.html

## 📱 기능 설명

### 1. 메인 페이지 (professor-main.html)

메인 페이지에는 다음 7개의 최신글 섹션이 표시됩니다:

1. **공지사항**: 최신 공지사항 5개
2. **한국사상과성경**: 관련 최신 글 5개
3. **세계사상과성경**: 관련 최신 글 5개
4. **책과논문**: 최신 저작물 5개
5. **열린마당**: 커뮤니티 글 5개
6. **게시판**: 일반 게시판 글 5개
7. **이미지&동영상**: 최신 미디어 6개

### 2. 로그인 시스템

- 이메일/아이디와 비밀번호로 로그인
- JWT 토큰 기반 인증
- 자동 로그인 유지 (7일)
- 역할별 접근 권한 제어

### 3. 회원가입

- 이름, 아이디, 이메일 필수
- 비밀번호 강도 표시
- 연락처 선택 사항
- 중복 아이디/이메일 검사

### 4. 관리자 대시보드

#### 통계 정보
- 총 회원수
- 총 게시글 수
- 각 게시판별 게시글 수
- 미디어 수량

#### 게시판 관리
관리자는 다음 게시판을 관리할 수 있습니다:

1. **공지사항**: 중요 공지사항 작성 및 관리
2. **한국사상과성경**: 관련 연구 자료 및 글
3. **세계사상과성경**: 비교 연구 자료
4. **책과논문**: 저작물 및 논문 목록
5. **열린마당**: 커뮤니티 게시판
6. **게시판**: 일반 게시판

#### CRUD 기능
- **생성 (Create)**: 새 글 작성
- **읽기 (Read)**: 글 목록 및 상세 보기
- **갱신 (Update)**: 기존 글 수정
- **삭제 (Delete)**: 글 삭제

#### 추가 기능
- 상단 고정 (Pin)
- 카테고리 분류
- 파일 첨부 지원
- 이미지/동영상 업로드
- 조회수 추적
- 작성자 정보 관리

## 🗄️ 데이터베이스 구조

### Collection (컬렉션)

1. **users** - 사용자 정보
   - username (아이디)
   - email (이메일)
   - password (비밀번호, 암호화)
   - name (이름)
   - phone (연락처)
   - role (역할: user/admin)

2. **boards** - 게시판 글
   - title (제목)
   - content (내용)
   - author (작성자)
   - boardType (게시판 타입)
   - category (카테고리)
   - tags (태그)
   - images (이미지 배열)
   - videoUrl (동영상 URL)
   - views (조회수)
   - isPinned (상단 고정 여부)
   - createdAt (생성일)
   - updatedAt (수정일)

3. **media** - 미디어 파일
   - title (제목)
   - description (설명)
   - mediaType (타입: image/video)
   - url (파일 URL)
   - thumbnail (썸네일)
   - uploadedBy (업로더)
   - category (카테고리)
   - views (조회수)

## 🚀 사용 방법

### 1. 첫 접속

1. 메인 페이지 접속: `professor-main.html`
2. 오른쪽 하단에서 "로그인" 클릭
3. 기본 관리자 계정으로 로그인

### 2. 게시글 작성

1. 관리자 대시보드 접속
2. 왼쪽 메뉴에서 원하는 게시판 선택
3. "글 쓰기" 버튼 클릭
4. 제목, 내용, 카테고리 입력
5. 필요시 상단 고정 체크
6. "저장" 클릭

### 3. 게시글 수정/삭제

1. 관리자 대시보드에서 해당 게시판 선택
2. 목록에서 "수정" 또는 "삭제" 버튼 클릭
3. 수정: 내용 변경 후 저장
4. 삭제: 확인 후 삭제

### 4. 미디어 업로드

1. 관리자 대시보드에서 "이미지&동영상" 선택
2. 업로드 버튼 클릭
3. 파일 선택 및 정보 입력
4. 업로드

## 🆘 문제 해결

### MongoDB 연결 오류
```bash
# MongoDB 설치 및 시작
sudo apt-get install mongodb
sudo systemctl start mongodb

# 또는 MongoDB Atlas 사용 (클라우드)
# .env 파일에 MONGODB_URI 설정
```

### 로그인 오류
1. 아이디/비밀번호 확인
2. 브라우저 캐시 초기화
3. 토큰 삭제 후 재로그인

### 파일 업로드 오류
1. 파일 크기 확인 (최대 10MB)
2. 파일 형식 확인 (이미지/동영상)
3. uploads 폴더 권한 확인
