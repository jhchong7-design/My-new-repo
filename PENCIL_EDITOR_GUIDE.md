## 🚀 How to Use

### Step 1: Login
1. Go to: `https://00too.app.super.myninja.ai/admin`
2. Login with your credentials
3. Navigate to any page

### Step 2: Enter Edit Mode
1. Look for the ✏️ pencil icon (top-right corner)
2. Click the pencil icon
3. Edit mode activates!

**What Happens:**
- Pencil icon becomes ✕ (close button)
- Admin sidebar appears on the right
- "✏️ Edit Mode Active" indicator appears top-left
- All text becomes editable

### Step 3: Edit Content
1. Click on any text you want to edit
2. Edit directly on the page
3. Use sidebar tools for formatting

**Formatting Options:**
```
[B] [I] [U] [S] | [H2] [H3] [P] [✕]
[●] [1.] [←] [↔] [→]
[🔗] [🖼️] [📁]
```

### Step 4: Save Changes
- Click "💾 Save Changes" in sidebar, OR
- Press `Ctrl+S`, OR
- Click away from edited text (auto-saves in 2 seconds)

### Step 5: Exit Edit Mode
- Click the ✕ button (top-right corner), OR
- Press `Escape`
